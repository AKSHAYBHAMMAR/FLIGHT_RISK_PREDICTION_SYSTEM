# Airlytics - Flight Risk Prediction System

A professional aviation-themed dashboard for predicting flight delays and assessing route risks using real-time weather and routing data.

## Features
- **Dynamic Route Mapping**: Visualize flight paths and waypoints globally.
- **Real-Time Weather Analytics**: Fetches current meteorological conditions for every flight segment.
- **Predictive Delay Modeling**: Estimates potential delays based on weather data and route distance.
- **Secure Authentication**: Signup/Login system with SHA-256 password hashing.
- **Data Export**: Export your flight analysis reports to CSV format.
- **Dashboard Visualization**: Interactive trend charts and risk distribution data.

## How to Run
Double-click the `run_airlytics.bat` file in the project folder to open the application in your default web browser.

## Technologies Used
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (via CDN)
- **Icons**: Lucide Icons
- **Mapping**: Leaflet.js (Map visualization)
- **Charts**: Chart.js (Data visualization)
- **API**: OpenWeatherMap (Weather & Geocoding), OSRM (Routing)
- **Database**: Mock authentication via LocalStorage (client-side)
